Sobre o projeto: Aplicativo mobile híbrido, desenvolvido com Ionic Framework e
Firebase Database.

Obs: O banco de dados utilizado (Firebase Database) é distribuído em nuvem,
portanto necessita de conexão com a internet para funcionar.

Como instalar dependências e executar:

1- Instalar o npm:

    https://nodejs.org/en/download/

2- Em um terminal, instalar o Ionic Framework, através do npm

    npm install -g ionic

3- Na pasta raiz do projeto, execute o seguinte comando:

    ionic serve

O projeto será carregado em uma aba do navegador padrão, em localhost.

Opcional:
O seguinte comando irá abrir o projeto em uma interface que simula a aparência e o
comportamento das plataformas Android e iOS:

    ionic lab

É provável que na primeira execução seja requisitado o download de alguns pacotes
através do npm, bastando confirmar seu download e instalação.
